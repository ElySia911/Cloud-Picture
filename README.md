# Cloud-Picture 云图库管理系统

## 项目简介

基于 Spring Boot + Redis + WebSocket 构建的云图库管理平台，
支持公共图库、私有图库以及团队共享图库。

## 技术栈

- SpringBoot
- MyBatisPlus
- Redis
- SaToken
- WebSocket
- Vue3
- TypeScript
- Ant Design Vue
- 腾讯云 COS 对象存储

## 核心功能

- 本地上传与URL上传，以图搜图，颜色搜图，抓取外部图库图片，AI编辑图片
- 公共图库
- 私有图库
- 团队共享图库
- 实时协同编辑
- 资源共享
- 可视化分析
